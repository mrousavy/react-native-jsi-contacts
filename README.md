# react-native-jsi-contacts

A contacts library for React Native using JSI

## Installation

```sh
npm install react-native-jsi-contacts
```

## Usage

```js
import { getContactsAsync } from "react-native-jsi-contacts";

const contacts = await getContactsAsync();
```
